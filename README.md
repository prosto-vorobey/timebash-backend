# Timebash.Backend

Backend-часть трекера пользовательской активности **Timebash**.  
Предоставляет REST API для управления журналами, активностями, категориями и сбора статистики.

## Стек

- **.NET 10** — платформа
- **ASP.NET Core** — Web API
- **Entity Framework Core** — ORM для PostgreSQL
- **PostgreSQL** — база данных
- **Docker / Docker Compose** — контейнеризация инфраструктуры
- **Swagger / OpenAPI** — документация API
- **Serilog + Seq** — структурированное логирование
- **FluentValidation** — валидация входящих данных
- **JWT** — аутентификация пользователей
- **xUnit** — модульное тестирование

## Архитектура

Проект разделён на слои с чётким разделением ответственности:

- **Timebash.Core** — доменные сущности, интерфейсы, DTO, исключения.
- **Timebash.Application** — бизнес-логика, сервисы, валидация (FluentValidation), расширения и классы-помощники.
- **Timebash.Infrastructure** — реализация репозиториев, EF Core, миграции.
- **Timebash.API** — ASP.NET Core Web API, контроллеры, JWT-аутентификация, middleware для обработки ошибок.

## Тестирование и покрытие

- Модульные тесты (xUnit) покрывают бизнес-логику:
  - Application layer: **99.8%**
  - Core layer: **85.7%**
- API и Infrastructure тестируются через интеграционные тесты (будут добавлены).

## Ближайшие планы

- Реализация endpoint'ов для сбора статистики по категориям.
- Наполнение интеграционных тестов.

## Связанные репозитории

- Timebash.Web (приватный) — фронтенд-часть приложения.
